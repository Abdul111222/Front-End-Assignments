Is javascript run in browser

Yes, JavaScript is executed in web browsers using built-in JavaScript engines.

Is javascript compiler or interpreter ?

JavaScript is both interpreted and compiled, as modern JavaScript engines use a hybrid approach called Just-In-Time (JIT) compilation, which combines aspects of both methods for better performance
